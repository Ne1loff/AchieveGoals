# AchieveGoals
Простой таск-трекер с упором на цели. Главной целью давнного проекта была разработка простого таск-трекера для достижения поставленых целей и задач. 

Планы на будущее:
- Добавление меток.
- Объединение задач в проекты.
- Создание командных проектов.

Используемые технологии:
- Kotlin
- Spring
- Svelte
- S3 Minio
- PostgreSQL
